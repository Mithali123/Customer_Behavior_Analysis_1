This project focuses on analyzing customer behavior using data analytics techniques. The goal is to extract meaningful insights from customer data by performing data cleaning, exploratory data analysis (EDA), SQL-based data querying, and building interactive dashboards using Power BI.
The project helps in understanding customer purchasing patterns, preferences, and trends to support better business decision-making.

Dataset
The dataset contains customer-related information such as:
Customer ID
Purchase Amount
Product Details
Review Ratings
Shipping Type
Discount Usage
Gender
The dataset is used to identify trends, customer preferences, and purchasing behavior.

Project Steps
Data Loading

Imported dataset using Python.
Verified data structure and data types.

Data Cleaning
Handled missing values.
Removed duplicates.
Standardized data formats.
Corrected inconsistent entries.

Exploratory Data Analysis (EDA)
Analyzed customer purchasing trends.
Studied the distribution of purchase amounts.
Compared review ratings across products.
Evaluated shipping and discount usage patterns.

SQL Analysis
Performed queries to extract key insights.
Identified top-selling products.
Compared purchase behavior across customer segments.
Analyzed average purchase amounts and rating patterns.

Power BI Dashboard
Created an interactive dashboard to visualize:
Customer purchase trends
Product performance
Revenue distribution
Customer segmentation
Shipping and discount impact

Results & Insights
Identified high-performing products based on customer ratings.
Observed purchase behavior differences across demographics.
Found a correlation between discount usage and purchase amount.
Provided data-driven insights to support marketing and sales strategies.
